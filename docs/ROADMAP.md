# ROADMAP

## Phase 1: MVP Calculator

Goal:
Build the first usable desktop-style trading tool.

Includes:

- Minimal landing screen
- Risk Calculator
- Daily Drawdown Calculator
- Sidebar navigation for the two MVP tools
- Guided wizard flow
- Local storage for calculator inputs
- Copy Summary
- Share Card modal
- PNG download
- Feedback button
- Helpful term explanations/tooltips
- Randomized Trading Insights
- Dark and light modes
- Theme preference persistence
- Geist typography
- Live animated atoms background
- Responsive landing and dashboard experience
- Calculation tests
- Compact centered dashboard layout
- Wide report layout instead of tall stacked cards
- Basic design system

Status:
Launch polish complete. Moving into final QA and launch preparation.
