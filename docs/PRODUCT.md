# PRODUCT

## Product Name

Prop Pilot

## Vision

Prop Pilot is a focused desktop-style trading tool that helps prop firm traders manage risk, understand rule limits, and make cleaner trading decisions before placing trades.

## Problem

Prop firm traders often fail challenges because they miscalculate position size, daily drawdown, max loss, or consistency rules. These mistakes are often operational rather than strategic.

## Target User

The primary user is a prop firm trader who needs fast, simple, reliable tools for challenge and funded account risk management.

## Value Proposition

Prop Pilot helps traders protect their account by turning prop firm rules into simple tools, clear reports, and fast risk signals.

## Core MVP

The current MVP includes:

- Minimal landing screen before the app dashboard
- Risk Calculator
- Daily Drawdown Calculator
- Guided wizard flow for each tool
- Sidebar dashboard
- Local storage for calculator inputs and preferences
- Copy Summary for shareable text output
- Share Card modal
- PNG download for branded report cards
- Feedback button
- Randomized Trading Insights for Daily Drawdown results
- Helpful term explanations/tooltips
- Dark mode and light mode
- Theme preference persistence
- Geist global typography
- Live animated "atoms" background
- Responsive dashboard and landing experience
- Tested calculation helpers

## Future Product Direction

Future versions may include:

- Consistency Calculator
- Prop Health Score
- Challenge Planner
- Presets
- Trading Journal
- Account tracking dashboard

Risk Calculator and Daily Drawdown should become separate sidebar tools instead of one giant combined wizard. Prop Health Score should remain the future signature feature once the core tools are stable.

## Current Product Flow

The current MVP flow is:

```txt
Landing Screen -> Start Calculating -> Dashboard with Sidebar -> Risk Calculator or Daily Drawdown Wizard -> Results -> Copy Summary or Share Card / PNG Download
```

The landing screen is intentionally minimal. It is not a full marketing website and should not grow into testimonials, pricing, blog sections, or long-form sales content during the MVP stage.
