# Prop Pilot Vision

## Mission

Prop Pilot exists to help prop traders make smarter, safer, and more consistent trading decisions. Its mission is to reduce avoidable rule violations by giving traders clear risk tools, transparent calculations, and calm decision support before they trade.

## Long-Term Vision

Over the next 3-5 years, Prop Pilot can become the daily operating system for prop traders.

The long-term vision is not just to offer a collection of calculators. Prop Pilot should become the place where traders plan sessions, check rule limits, track challenge progress, review risk, and build better trading discipline.

Prop Pilot should help traders answer important questions quickly:

- Can I take this trade?
- How much can I risk?
- Am I close to a rule breach?
- Am I trading consistently?
- What should I focus on today?

## Core Principles

- Simplicity over complexity
- Accuracy before features
- Speed and clarity
- Trust through transparent calculations
- Mobile-first experience
- Professional design
- Education alongside tools
- Incremental improvement

## User Experience Goals

Prop Pilot should help users feel:

- Calm
- Confident
- In control
- Focused
- Never overwhelmed

The app should reduce stress, not add more decisions. Every screen should help traders understand their risk position quickly.

## Product Philosophy

Every feature in Prop Pilot should solve a real trading problem.

Features should only be added when they increase user value without creating unnecessary complexity. A simple tool that traders trust is more valuable than a crowded dashboard that tries to do everything.

Prop Pilot should grow carefully. Each new tool should be accurate, understandable, and useful before another feature is added.

## What We Will Build

Prop Pilot may grow to include:

- Risk management tools
- Daily drawdown tools
- Consistency tracking
- Prop firm rule engine
- Trading dashboards
- Performance analytics
- Planning tools
- Educational insights
- Prop Health Score as a signature account safety signal

## What We Will Avoid

Prop Pilot should avoid:

- Feature bloat
- Complex interfaces
- Unverified calculations
- Copying competitors without purpose
- Adding features solely because they are trendy
- Hiding important assumptions
- Making traders depend on unclear or overly complex outputs

## Success Definition

Success is bigger than revenue.

Prop Pilot is successful if:

- Traders use it before every trading session.
- It helps traders avoid rule violations.
- It builds trust through reliable calculations.
- It becomes a respected tool in the prop trading community.
- It helps traders become more disciplined and consistent.

## Guiding Statement

Every decision made in Prop Pilot should support one goal: helping traders make smarter, safer, and more consistent trading decisions.
