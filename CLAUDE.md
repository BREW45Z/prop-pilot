# Project Rules

This is a Next.js trading calculator application.

Priorities:

1. Build calculators first.
2. No authentication.
3. No database.
4. No Stripe.
5. No dashboards.
6. No API integrations.

Current goal:
Build a Consistency Calculator.

Folder responsibilities:

- app = pages
- components = UI
- lib = calculation logic
- types = interfaces

Always:
- Keep code simple
- Use TypeScript
- Reuse existing components
- Explain file changes before making them

Never:
- Create unnecessary files
- Refactor unrelated code
- Add features not requested
